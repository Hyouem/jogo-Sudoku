# Jogo de Sudoku em Java

Projeto desenvolvido como desafio prático da **Digital Innovation One (DIO)**, com o objetivo de aplicar conceitos de **Programação Orientada a Objetos**, **boas práticas de código** e **versionamento com Git/GitHub**.

---

## Tecnologias Utilizadas
- Java 17+
- Git e GitHub
- Programação Orientada a Objetos (POO)
- Execução via Terminal
- (Opcional) Interface Gráfica

---

## Funcionalidades
- Inicialização do tabuleiro via argumentos de execução
- Validação de jogadas
- Identificação de células fixas e editáveis
- Exibição do tabuleiro no terminal
- Verificação de vitória
